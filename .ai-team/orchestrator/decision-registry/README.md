# 决策记录库

> 存储所有关键决策，支持追溯和复盘

## 决策记录格式

```markdown
## ADR-XXX: 决策标题

**日期**: YYYY-MM-DD
**状态**: `proposed` | `accepted` | `deprecated` | `superseded`
**决策者**: [expert-1, expert-2, ...]
**影响范围**: [模块/系统]

### 背景
描述决策背景和问题

### 决策
描述做出的决策

### 理由
解释为什么做出这个决策

### 后果
描述决策的影响和后果

### 相关任务
- TASK-XXX
- TASK-YYY
```

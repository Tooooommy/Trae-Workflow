# 机器学习模式

> 构建可扩展、可维护的机器学习系统

## 核心概念

### ML生命周期

```mermaid
flowchart LR
    A[数据准备] --> B[特征工程]
    B --> C[模型训练]
    C --> D[模型评估]
    D --> E[模型部署]
    E --> F[监控优化]
    F --> B
```

### ML系统组件

| 组件 | 说明 |
|------|------|
| 数据管道 | 数据采集、清洗、存储 |
| 特征存储 | 特征定义、计算、服务 |
| 模型注册 | 模型版本、元数据管理 |
| 服务系统 | 模型推理、API服务 |
| 监控系统 | 性能、漂移检测 |

## 特征工程

### 特征类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 数值型 | 连续/离散数值 | 年龄、价格 |
| 类别型 | 离散类别 | 性别、城市 |
| 文本型 | 文本数据 | 评论、描述 |
| 时间型 | 时间相关 | 时间戳、周期 |

### 特征处理

| 技术 | 说明 |
|------|------|
| 归一化 | 缩放到[0,1] |
| 标准化 | 均值0方差1 |
| 编码 | One-Hot、Label、Target |
| 嵌入 | Word2Vec、Embedding |

### 特征选择

| 方法 | 说明 |
|------|------|
| 过滤法 | 统计特征重要性 |
| 包装法 | 递归特征消除 |
| 嵌入法 | L1正则、树模型重要性 |

## 模型训练

### 训练策略

| 策略 | 说明 | 适用场景 |
|------|------|----------|
| 批量训练 | 全量数据训练 | 数据量小 |
| 增量训练 | 持续更新模型 | 数据持续产生 |
| 分布式训练 | 多机多卡训练 | 大模型、大数据 |

### 超参数调优

| 方法 | 说明 |
|------|------|
| 网格搜索 | 遍历所有组合 |
| 随机搜索 | 随机采样组合 |
| 贝叶斯优化 | 智能搜索策略 |
| Hyperband | 资源自适应 |

### 防止过拟合

| 技术 | 说明 |
|------|------|
| 正则化 | L1/L2正则 |
| Dropout | 随机丢弃神经元 |
| 早停 | 验证集性能不再提升时停止 |
| 数据增强 | 扩充训练数据 |

## 模型评估

### 分类指标

| 指标 | 说明 | 公式 |
|------|------|------|
| 准确率 | 正确预测比例 | (TP+TN)/(TP+TN+FP+FN) |
| 精确率 | 预测为正中真正为正 | TP/(TP+FP) |
| 召回率 | 真正为正中被预测为正 | TP/(TP+FN) |
| F1 | 精确率和召回率调和平均 | 2*P*R/(P+R) |
| AUC-ROC | 分类能力综合指标 | ROC曲线下面积 |

### 回归指标

| 指标 | 说明 |
|------|------|
| MAE | 平均绝对误差 |
| MSE | 均方误差 |
| RMSE | 均方根误差 |
| R² | 决定系数 |

### 交叉验证

| 方法 | 说明 |
|------|------|
| K折 | 分K份轮流验证 |
| 留一 | 每次留一个样本 |
| 时间序列 | 按时间划分 |

### 部署模式

| 模式 | 说明 | 适用场景 |
|------|------|----------|
| 批处理 | 离线批量预测 | 报表、分析 |

### 漂移检测

```python
def detect_drift(baseline, current, threshold=0.05):
    drift_score = kl_divergence(baseline, current)
    return drift_score > threshold
```

## 反模式

| 反模式 | 问题 | 解决方案 |
|--------|------|----------|
| 训练服务不一致 | 离线在线特征差异 | 统一特征管道 |
| 模型腐烂 | 性能随时间下降 | 持续监控重训 |
| 数据泄露 | 特征包含标签信息 | 严格特征审查 |
| 过度工程 | 复杂模型难维护 | 简单模型优先 |

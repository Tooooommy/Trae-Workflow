# MVP技术栈

## 全栈方案（推荐用于快速验证）

### NextJS

```yaml
核心框架:
  - Next.js (React 全栈框架)
  - TypeScript (严格模式)
  - Shadcn/ui (组件库)
  - Tailwind CSS (样式库)
  - Supabase (数据库)
  - Vercel (部署平台)
```

### RemixJS

```yaml
核心框架:
  - Remix (React 全栈框架)
  - TypeScript (严格模式)
  - Postgres (主数据库)
  - Prisma (ORM + 迁移)
  - Railway (部署平台)
  - Shadcn/ui (组件库)
  - Tailwind CSS (样式库)
  - Pino (日志)
  - Redis (缓存/会话)
```

### Shopify APP

```yaml
核心框架:
  - React Router (全栈框架)
  - shopify-app-template-react-router (Shopify APP 模板)
  - TypeScript (严格模式)
```

## 移动端

### React Native

```yaml
核心框架:
  - Ignite (脚手架工具)
  - Tamagui (UI框架)
```

## 前端

### React

```yaml
核心框架:
  - Vite (前端构建工具)
  - React 18 (核心框架)
  - TypeScript (严格模式)
  - React Router DOM (v6+)
  - Zustand (状态管理)
  - React Query (服务端状态)
  - Shadcn/ui + Radix UI (组件库)
  - Tailwind CSS (样式库)
```

## 后端

### Python

```yaml
框架层:
  - FastAPI (核心框架)
  - Pydantic v2 (数据验证 + 配置管理)
  - PostgreSQL (主数据库)
  - TortoiseORM (异步ORM)
  - Aerich (双迁移方案)
  - Redis (缓存/会话)
  - Celery + RabbitMQ (异步任务)
  - Structlog (结构化日志)
  - Swagger (API文档)
  - Pytest (测试框架)
```

### Node

```yaml
框架层:
  - HonoJS (核心框架)
  - Typescript (类型系统)
  - PostgreSQL (主数据库)
  - Prisma (ORM + 迁移)
  - Redis (缓存/会话/队列)
  - BullMQ (消息队列)
  - Pino (结构化日志)
  - Swagger (API文档)
  - Jest (测试框架)
```

## 1. 交互原则

| 原则 | 说明 |
|------|------|
| 反馈及时 | 操作后立即反馈 |
| 一致性 | 相同操作相同反馈 |
| 可预测 | 用户能预知结果 |
| 可撤销 | 支持撤销操作 |
| 容错性 | 防止误操作 |

## 2. 手势交互

### 2.1 触摸手势

| 手势 | 操作 | 用途 |
|------|------|------|
| 点击 | Tap | 选择/确认 |
| 长按 | Long Press | 显示菜单 |
| 滑动 | Swipe | 切换/滚动 |
| 双击 | Double Tap | 放大 |
| 捏合 | Pinch | 缩放 |

### 2.2 手势区域

| 区域 | 手势 | 操作 |
|------|------|------|
| 左边缘 | 右滑 | 返回 |
| 右边缘 | 左滑 | 打开菜单 |
| 列表项 | 左滑 | 删除/更多 |

## 3. 动效规范

### 3.1 动效时长

| 类型 | 时长 | 用途 |
|------|------|------|
| 微交互 | 100-150ms | 按钮点击 |
| 状态切换 | 200-300ms | 展开/收起 |
| 页面转场 | 300-500ms | 页面跳转 |
| 复杂动画 | 500-1000ms | 引导动画 |

### 3.2 缓动函数

| 缓动 | 曲线 | 用途 |
|------|------|------|
| ease-out | 快→慢 | 进入动画 |
| ease-in | 慢→快 | 退出动画 |
| ease-in-out | 慢→快→慢 | 往返动画 |
| spring | 弹性 | 强调效果 |

### 3.3 动效类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 淡入淡出 | opacity 变化 | 弹窗显示 |
| 滑动 | transform 移动 | 抽屉打开 |
| 缩放 | transform scale | 按钮点击 |
| 翻转 | transform rotate | 卡片翻转 |

## 4. 状态反馈

### 4.1 加载状态

| 状态 | 表现 | 时长 |
|------|------|------|
| 即时加载 | 骨架屏 | < 1s |
| 短时加载 | 加载动画 | 1-3s |
| 长时加载 | 进度条 | > 3s |

### 4.2 操作反馈

| 操作 | 反馈 | 形式 |
|------|------|------|
| 成功 | 成功提示 | Toast |
| 失败 | 错误提示 | Toast + 详情 |
| 警告 | 警告提示 | Toast |
| 确认 | 确认弹窗 | Modal |

### 4.3 空状态

| 场景 | 表现 |
|------|------|
| 无数据 | 空状态插图 + 引导文案 |
| 无网络 | 网络错误插图 + 重试按钮 |
| 搜索无结果 | 无结果插图 + 建议 |

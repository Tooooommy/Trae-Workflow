## 3. CI/CD 配置

### 3.1 流水线

```
代码提交 → 构建 → 测试 → 部署 → 验证
```

### 3.2 触发条件

| 分支 | 触发 | 部署环境 |
|------|------|----------|
| develop | push | 开发 |
| staging | push | 测试 |
| main | push | 生产 |

### 3.3 构建命令

```bash
# 构建
npm run build

# Docker 构建
docker build -t app:latest .

# 推送镜像
docker push registry/app:latest
```

## 4. 部署步骤

### 4.1 部署前检查

- [ ] 代码已合并
- [ ] 测试已通过
- [ ] 镜像已构建
- [ ] 配置已更新

### 4.2 部署命令

```bash
# Kubernetes
kubectl apply -f k8s/

# Docker Compose
docker-compose up -d
```

### 4.3 部署验证

```bash
# 健康检查
curl http://localhost:8080/health

# 日志查看
kubectl logs -f deployment/app
```

## 6. 回滚

### 6.1 回滚命令

```bash

# Docker

```

# Git 规范

## 提交格式

```
<type>(<scope>): <subject>
```

## 类型

| 类型     | 说明     |
| -------- | -------- |
| feat     | 新功能   |
| fix      | 修复 bug |
| docs     | 文档     |
| style    | 格式     |
| refactor | 重构     |
| test     | 测试     |
| chore    | 工具     |
| perf     | 性能     |

## 主题

- 动词-名词
- ≤ 50 字符

## PR

1. 分析提交历史
2. git diff 查看更改
3. 起草 PR 摘要
4. 包含测试计划

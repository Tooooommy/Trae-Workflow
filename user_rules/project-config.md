> 此文件定义项目特定的配置，优先级最高。

## 技术栈选择

无指定时使用以下默认技术栈：

| 层级 | 默认选择 |
|------|----------|
| 前端 | React/Vue3 + TypeScript + Vite |
| 后端 | Node.js (Nest.js) / Python (FastAPI) |
| 数据库 | PostgreSQL/MySQL |

---

## 性能目标

| 指标 | 目标值 |
|------|--------|
| API 响应时间 | < 200ms (P95) |
| 错误率 | < 0.1% |
| 可用性 | > 99.9% |

---

## 需求确认

开始任何任务前，必须确认：

1. **业务目标** - 明确要解决的问题
2. **技术栈约束** - 确认使用的技术栈
3. **性能需求** - 确认性能指标要求
4. **安全需求** - 确认安全级别要求

> 模糊需求需提供 2-3 个对比方案供选择。

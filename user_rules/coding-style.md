# 代码规范

## 原则

| 原则 | 说明 |
|------|------|
| 不可变性 | 创建新对象，不修改现有 |
| 文件组织 | 200-400 行，最多 800 |
| 错误处理 | 每层处理，UI 友好 |
| 输入验证 | 边界验证，快速失败 |

## 格式

| 配置 | 值 |
|------|-----|
| 缩进 | 2 空格 |
| 行长度 | ≤ 100 |
| 引号 | 单引号 |

## 命名

| 类型 | 规则 | 示例 |
|------|------|------|
| 组件 | PascalCase | UserProfile.tsx |
| 工具 | kebab-case | user-utils.ts |

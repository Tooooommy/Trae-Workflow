# Hooks 系统

## Hook 类型

| Hook        | 说明                   |
| ----------- | ---------------------- |
| PreToolUse  | 工具执行前（验证、修改） |
| PostToolUse | 工具执行后（格式化、检查） |
| Stop        | 会话结束（最终验证）   |

## 自动接受权限

- 受信任、明确的计划可启用
- 探索性工作禁用
- 禁用 dangerously-skip-permissions

## TodoWrite 最佳实践

- 跟踪多步骤任务进度
- 验证指令理解
- 展示实现步骤

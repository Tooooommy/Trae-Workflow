# 核心原则

> 优先级最高

## 五条原则

| #   | 原则         | 说明                       |
| --- | ------------ | -------------------------- |
| 1   | 智能体优先   | 委托专业智能体处理复杂任务 |
| 2   | 测试驱动     | 先写测试，80%+ 覆盖率      |
| 3   | 安全第一     | 验证所有输入，保护敏感数据 |
| 4   | 不可变性     | 创建新对象，不修改现有     |
| 5   | 先规划后执行 | 复杂功能先规划再实现       |

## 示例

```
WRONG:  modify(obj, 'field', value)
CORRECT: update(obj, 'field', value)
```

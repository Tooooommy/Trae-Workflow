# 安全规范

## 提交前检查

- [ ] 无硬编码密钥
- [ ] 输入已验证
- [ ] 防止 SQL 注入
- [ ] 防止 XSS
- [ ] 启用 CSRF 保护
- [ ] 认证/授权已验证
- [ ] 端点已限速
- [ ] 错误消息不泄露敏感数据

## 密钥管理

- **禁止硬编码**
- **环境变量** - 使用 env 或密钥管理器
- **启动验证** - 启动时验证密钥
- **立即轮换** - 发现泄露立即轮换

## 问题处理流程

```
发现问题 → 停止 → security-reviewer → 修复 CRITICAL → 轮换密钥 → 审查类似问题
```

## 输入验证

- 系统边界验证
- 基于模式验证
- 快速失败
- 绝不信任外部数据

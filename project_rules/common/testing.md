# 通用测试规范

> 所有项目通用的测试规范。

## 测试类型

| 类型 | 范围 |
|------|------|
| 单元测试 | 单个函数、组件 |
| 集成测试 | API、数据库 |
| E2E 测试 | 关键用户流程 |

## 覆盖率要求

- **最低**：80%
- **目标**：90%+

## TDD 工作流

```
1. RED   → 测试失败
2. GREEN → 最小实现
3. IMPROVE → 重构
```

## 测试隔离

- 每个测试独立运行
- 不依赖执行顺序
- 使用 mock 隔离外部依赖
